### Backend Processing
```
1. Receive request
2. Validate message exists
3. Preprocess Urdu text
   - Remove extra spaces
   - Normalize Unicode
   - Tokenize
4. Run AI model
   - Convert to embeddings
   - Model inference
   - Get probability score
5. Format response
   - Determine classification
   - Calculate confidence
   - Generate explanation
```
